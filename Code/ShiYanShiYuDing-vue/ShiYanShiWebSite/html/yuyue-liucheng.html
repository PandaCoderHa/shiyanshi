开始预约

帮助与说明

1 预约流程

1.1 首先选择实验室

  • 点击对应需要预约的各实验室
  • 点击查看更多实验室可以看到所有可预约的实验室

1.2 选择预约时间

  • 预约时间可选择10天之内，点击预约
  • 标题注意：当日不可用时间，允许时长
  • 填写预约用途， 必须填写：课程相关实验项目名称， 选填：指导教师
  • 选择预约时间段，开始时间不能小于开放时间，预约结束时间不能大于开放时间
  • 教师可以手动选择当前课程下所有的学生，可以根据学号搜索学生（教师预约人数不得少于10人）
  • 最后点击提交申请（预约记录中可查询信息）
