开始预约

帮助与说明

1 注册登陆

1.1 校内用户

  • 校内用户请使用统一认证系统账号、密码进行登陆

1.2 校外用户

  • 校外用户请先申请取得学校统一认证系统账号，再使用统一认证系统账号、密码进行登陆

2 预约流程

2.1 选择会议室或者实验室

点击对应需要预约的会议室或者实验室

2.2 填写预约用途和时间

填写预约用途

填写预约时间，开始时间不能小于当前时间，预约结束时间不能小于开始时间

2.3 选择预约项目（预约会议室无须考虑此项）

  • 预约整个实验室

    选择独占实验室则表示其他人不能在时间段内预约实验室或实验室内的项目

  • 预约实验室项目
    1. 选座形式，根据实验室地图映射，选择对应位置的项目
    2. 列表形式，在实验室项目列表内选择对应的项目

3 反馈

填写对应的标题、邮箱和表述进行反馈
